# Default Keymap for CarlosEDP's 60% XD60 PCB

Custom layout with some custom layers. Mac friendly.

## Layout

[Carlosedp's Layout](https://imgur.com/a/gz5nD6y)

## Additional Notes

Based on the default Keymap for XD60.

## Build

To build the keymap, simply run `make xd60/rev3:carlosedp` or build in Docker with `util/docker_build.sh xd60/rev3:carlosedp`
